# Lab 04 - Kubernetes para Analitica

## Objetivo

Ejecutar procesos batch con Job y CronJob, con configuracion externa.

## Prerrequisitos

- Haber completado `labs/00-prework`.
- Tener cluster `kind` activo y accesible por `kubectl`.

## Pasos guiados

1. Crear y ejecutar un Job ETL.
2. Crear un CronJob de ejecucion periodica.
3. Inyectar configuracion con ConfigMap.
4. Inyectar credenciales con Secret.
